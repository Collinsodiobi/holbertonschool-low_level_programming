# Simple Calculator

## Compilation
gcc -Wall -Werror -Wextra -pedantic -std=gnu89 calculator.c -o calculator

## Run
./calculator

## Description
This program is a simple terminal-based calculator.
It displays a menu and allows the user to choose an operation.

## Supported operations
- Add
- Subtract
- Multiply
- Divide

## Notes
- Uses scanf for input
- Handles division by zero
